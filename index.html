Register

  • Url
    • /register
  • Method
    • POST
  • Request Body
    • name as string
    • email as string, must be unique
    • password as string, at least 6 characters
  • Response

      "_id": 62991b054e7e8ee1d59b32d3,
      "error": false,
      "message": "User created"
    }
